**Concepto**: Confías en tus sentidos afinados y tu capacidad de observación más que en cualquier otra cosa. Cada detalle que otros pasan por alto es una pista para ti. Detectas mentiras con una mirada, notas lo que está fuera de lugar y usas tu percepción sobrehumana para desentrañar cualquier misterio.

## Estilo de Juego

El Investigador Empirista es el observador supremo:

1. **Percepción sobrehumana**: [Pelear a Ciegas](/dotes/investigador/pelear-a-ciegas/) + [Sentir lo Invisible](/dotes/investigador/sentir-lo-invisible/)
2. **Detección de mentiras**: Metodología + [Detector de Mentiras](/dotes/investigador/detector-de-mentiras/)
3. **Soporte táctico**: [Ojo del Empirista](/dotes/investigador/ojo-del-empirista/) + [Golpe Didáctico](/dotes/investigador/golpe-didactico/)
4. **Defensa perceptiva**: [Prever el Peligro](/dotes/investigador/prever-el-peligro/)

### Beneficio de Metodología
El Empirismo permite usar Percepción para:
- Intuir Motivación
- Percibir engaños
- Detectar mentiras

### Secuencia de Combate
1. [Idear Estratagema](/dotes/investigador/idear-una-estratagema/) (gratis con investigación)
2. Golpe con INT a daño
3. [Ojo del Empirista](/dotes/investigador/ojo-del-empirista/) para señalar y dar desprevenido
4. [Prever el Peligro](/dotes/investigador/prever-el-peligro/) como defensa

### Fortalezas
- Excelente contra invisibles y escondidos
- Detecta mentiras con facilidad
- Soporte táctico para el grupo
- Defensa basada en Percepción

### Debilidades
- Menos daño que otras metodologías
- Depende de altos modificadores de Percepción
- Menos útil contra enemigos honestos
