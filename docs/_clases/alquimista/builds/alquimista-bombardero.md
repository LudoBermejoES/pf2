# Alquimista Bombardero

**Concepto**: Eres un maestro de las explosiones alquímicas, un artillero que llueve destrucción sobre sus enemigos. Tu laboratorio es un polvorín de bombas de todo tipo: fuego, ácido, electricidad, frío. Con precisión letal, lanzas tus creaciones para explotar justo donde más duele.

## Estilo de Juego

El Alquimista Bombardero es artillería de área:

1. **Daño de área**: Salpicadura afecta múltiples enemigos
2. **Debuffs**: [Bomba Debilitante]({{ '/clases/alquimista/dotes/#bomba-debilitante' | relative_url }}) añade condiciones
3. **Versatilidad elemental**: Diferentes bombas para diferentes resistencias
4. **Precisión**: [Bombas Infalibles]({{ '/clases/alquimista/dotes/#bombas-infalibles' | relative_url }}) ignora cobertura

### Tipos de Bombas
| Bomba | Daño | Efecto |
|-------|------|--------|
| Fuego de alquimista | Fuego | Persistente |
| Frasco de ácido | Ácido | Persistente |
| Relámpago embotellado | Electricidad | Desprevenido |
| Bomba de escarcha | Frío | Penalizador velocidad |
| Bomba de trueno | Sónico | Ensordece |

### Secuencia de Combate
1. [Bombardero Rápido]({{ '/clases/alquimista/dotes/#bombardero-rapido' | relative_url }}) (saca + lanza)
2. Segunda bomba normal
3. Tercera bomba o reposicionamiento
4. Usa [Bomba Debilitante]({{ '/clases/alquimista/dotes/#bomba-debilitante' | relative_url }}) en objetivos prioritarios

### Salpicadura Expandida (Nivel 10+)
- Radio aumenta a 3m
- +INT al daño de salpicadura
- Con descubrimiento de bombardero: 2x INT a salpicadura

### Fortalezas
- Excelente daño de área
- Versatilidad elemental
- Debuffs poderosos
- Alcance seguro

### Debilidades
- Recursos limitados (bombas/día)
- Menos efectivo 1v1
- Cuidado con el fuego amigo
- Requiere preparación diaria
