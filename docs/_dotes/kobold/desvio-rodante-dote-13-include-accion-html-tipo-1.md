## Desvío rodante · Dote 13 · {% include accion.html tipo="1" %}

<div class="feat-traits-header" markdown="0"><a href="/apendices/rasgos/kobold/" class="feat-trait">Kobold</a></div>

**Requisitos:** experto en Acrobacias y Engaño

Intenta Rodar a través del espacio de un oponente. Si tienes éxito o éxito crítico y no terminas tu movimiento adyacente a ese oponente, puedes intentar Crear una distracción para distraer a ese oponente. Obtienes un bonificador +1 por circunstancia a la prueba de Engaño (o un bonificador +2 por circunstancia si obtuviste éxito crítico en la prueba de Acrobacias para Rodar a través) para Crear una distracción. Si tienes éxito o éxito crítico al Crear una distracción, pasas a estar oculto únicamente para la criatura cuyo espacio rodaste a través.
