## Oportunista rodante · {% include accion.html tipo="libre" %}

<div class="feat-traits-header" markdown="0"><a href="/apendices/rasgos/arquetipo/" class="feat-trait">Arquetipo</a><a href="/apendices/rasgos/ataque/" class="feat-trait">Ataque</a></div>

**Prerrequisitos:** Dedicación de Acróbata

**Frecuencia:** una vez por minuto

**Requisitos:** Tu acción más reciente fue Rodar a través o Golpe rodante, y te moviste con éxito a través del espacio de un enemigo.

Usas un estallido de resistencia para realizar una acrobacia impresionante mientras atraviesas el espacio de un enemigo, dejándolo tumbado de espaldas. Intentas Derribar al enemigo cuyo espacio atravesaste. Puedes usar Acrobacias en lugar de Atletismo para esta prueba.
