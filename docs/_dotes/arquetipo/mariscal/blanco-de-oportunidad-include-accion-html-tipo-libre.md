## Blanco de oportunidad · {% include accion.html tipo="libre" %}

<div class="feat-traits-header" markdown="0"><a href="/apendices/rasgos/arquetipo/" class="feat-trait">Arquetipo</a><a href="/apendices/rasgos/manipular/" class="feat-trait">Manipular</a></div>

**Prerrequisitos:** Dedicación de Mariscal

**Desencadenante:** Un aliado tiene éxito con un Golpe a distancia contra un oponente en el primer incremento de alcance de tu arma.

**Requisitos:** Tienes un arma a distancia o arrojadiza en la mano.

Aprovechas el ataque de tu aliado y utilizas la oportunidad para asegurar un golpe propio, sumándote a la andanada. Realiza un Golpe a distancia con penalización –2 contra el oponente objetivo del ataque desencadenante. Este Golpe no cuenta para tu penalización por ataques múltiples, y tu penalización por ataques múltiples no se aplica a este Golpe. Si este Golpe tiene éxito, combina el daño de tu ataque con el daño del ataque de tu aliado a efectos de resistencias y debilidades.
