## Blitz de armas dobles · {% include accion.html tipo="2" %}

<div class="feat-traits-header" markdown="0"><a href="/apendices/rasgos/arquetipo/" class="feat-trait">Arquetipo</a></div>

**Prerrequisitos:** Dedicación de Guerrero de armas dobles

**Requisitos:** Estás empuñando dos armas de melé de una mano, cada una en una mano distinta.

Atacas mientras te abres paso por el campo de batalla. Avanza hasta tu Velocidad. Durante este movimiento, puedes Golpear una vez con cada una de las dos armas de melé de una mano. Cada uno de estos Golpes puede realizarse en cualquier punto durante tu movimiento.
