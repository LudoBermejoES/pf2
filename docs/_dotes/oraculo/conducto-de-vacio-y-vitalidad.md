## Conducto de Vacio y Vitalidad

**Prerrequisitos:** cualquier misterio de oraculo

**Requisitos:** Tienes sanar o danar como hechizo insignia y un espacio de hechizo disponible para lanzarlo.

Usas la energia inestable de tu maldicion para manipular la magia divina mas basica. Lanzas un hechizo sanar o danar de 3 acciones, gastando el espacio como es normal. Si el hechizo restaura Puntos de Golpe a una o mas criaturas, entonces una criatura sanada por este hechizo recupera un numero de Puntos de Golpe adicionales igual a 1d8 x tu valor de vinculado a la maldicion; si el hechizo dana a una o mas criaturas, entonces una criatura danada por este hechizo recibe dano adicional igual a 1d8 x tu valor de vinculado a la maldicion.
