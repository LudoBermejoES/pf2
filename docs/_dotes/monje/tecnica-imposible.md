## Tecnica Imposible {% include accion.html tipo="reaccion" %}

**Dote 20**

*Fortuna, Monje*

**Desencadenante** El ataque de un enemigo te impacta o fallas una salvacion contra la habilidad de un enemigo.

**Requisitos** No llevas armadura ni estas fatigado.

Ejecutas una maniobra que desafia la posibilidad. Si el efecto desencadenante fue el ataque de un enemigo impactandote, el enemigo vuelve a tirar la tirada de ataque y usa el resultado menor. Si el efecto desencadenante fue que fallaste una salvacion, vuelves a tirar la salvacion y usas el resultado mayor.
