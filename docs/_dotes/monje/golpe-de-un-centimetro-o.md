**Prerrequisitos** golpes expertos

Pones toda tu fuerza en un unico golpe poderoso y cuidadosamente controlado. Haz un Golpe desarmado. Si gastas dos acciones y este Golpe impacta, infliges un dado adicional de dano de arma. Puedes en su lugar gastar 3 acciones para realizar un ataque aun mas poderoso, infligiendo un segundo dado adicional de dano de arma en un impacto.

Si eres de al menos nivel 10, el numero de dados adicionales que agregas con esta dote se duplica, para un total de 2 dados adicionales si gastas 2 acciones o 4 dados adicionales si gastas 3 acciones. Si eres de al menos nivel 18, el numero de dados adicionales que agregas con esta dote se triplica, para un total de 3 dados adicionales si gastas 2 acciones o 6 dados adicionales si gastas 3 acciones.
