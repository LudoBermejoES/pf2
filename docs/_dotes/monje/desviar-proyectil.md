**Desencadenante** Eres el objetivo de un ataque fisico a distancia.

**Requisitos** Eres consciente del ataque, no estas desprevenido contra el y tienes una mano libre.

Obtienes un bonificador de circunstancia +4 a la CA contra el ataque desencadenante. Si el ataque falla, lo has desviado. No puedes usar esta dote para desviar proyectiles a distancia inusualmente masivos (como rocas o proyectiles de balista).
