## Sombras distractoras

<div class="feat-traits-header" markdown="0"><a href="/apendices/rasgos/mediano/" class="feat-trait">Mediano</a></div>

**Dote 1** · Mediano

Has aprendido a permanecer oculto usando a la gente grande como distracción, y así no atraer la atención hacia ti.

**Beneficio:** Puedes utilizar criaturas por lo menos una categoría de tamaño mayores que tú (normalmente Medianas o mayores) como cobertura para las acciones Esconderse y Movimiento furtivo, aunque sigues sin poder utilizar a dichas criaturas como cobertura para otros usos, como por ejemplo para la acción Ponerse a cubierto.
