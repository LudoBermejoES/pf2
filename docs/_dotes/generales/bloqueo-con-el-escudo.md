**Desencadenante:** Podrías sufrir daño debido a un ataque físico (contundente, perforante o cortante) debido a un ataque cuando tienes el escudo alzado.

Interpones tu escudo para protegerte de un Golpe. El escudo evita que sufras una cantidad de daño igual a la Dureza del mismo. Ambos os repartís el daño sobrante, lo que posiblemente puede romper o destruir el escudo.
