**Prerrequisitos:** entrenado en Ocultismo

Los miembros de tu culto frecuentemente se hacen pasar por adoradores de otras religiones. Puedes usar Ocultismo en lugar de Engaño para Hacerte pasar por un adorador típico de otra fe o para Mentir específicamente para afirmar que eres miembro de la fe que estás suplantando. Todavía necesitas usar la habilidad Engaño para Hacerte pasar por un adorador específico o para realizar otras acciones engañosas, como intentar Mentir sobre cualquier otro asunto.
