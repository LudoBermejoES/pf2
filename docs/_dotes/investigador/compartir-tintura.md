**Prerrequisitos:** metodologia de ciencias alquimicas

Usas Tintura Rapida, y el objeto que creas permanece potente hasta el comienzo de tu siguiente turno. Entonces puedes Interactuar para pasar el objeto a otra criatura. Si lo pasas lanzandolo, tu ataque a distancia tiene exito automaticamente. Debes tener un vial versatil para usar Tintura Rapida, como es normal.
