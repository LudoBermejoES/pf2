# Detective

**Bagaje**

Resolviste delitos como inspector de policía o bien aceptaste trabajos de clientes ricos como investigador privado. Podrías haberte convertido en aventurero como parte de tu próximo gran misterio, pero probablemente fuera debido a las consecuencias o al desenlace de un caso anterior.

**Mejoras de atributo:** Una a Inteligencia o Sabiduría, otra gratuita

**Habilidades:** Sociedad, Saber (bajos fondos)

**Dote:** Callejeo
