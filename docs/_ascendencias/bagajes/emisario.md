# Emisario

**Bagaje**

Como diplomático o mensajero, has viajado por todas partes. Comunicarte con gente nueva y tejer alianzas eran tus puntos fuertes.

**Mejoras de atributo:** Una a Inteligencia o Carisma, otra gratuita

**Habilidades:** Sociedad, Saber (una ciudad que has visitado a menudo)

**Dote:** Multilingüe
