# Adivino

**Bagaje**

Las hebras del Destino son claras para ti, puesto que has aprendido muchas de las formas tradicionales por las que los legos pueden adivinar el futuro. Podrías haber utilizado dichas habilidades para guiar a tu comunidad, o simplemente para ganar dinero. Pero incluso el más leve atisbo de estas prácticas te conecta con los misterios ocultos del universo.

**Mejoras de atributo:** Una a Inteligencia o Carisma, otra gratuita

**Habilidades:** Ocultismo, Saber (buenaventura)

**Dote:** Identificación de rarezas
