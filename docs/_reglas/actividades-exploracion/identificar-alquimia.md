Utilizas la [Artesanía]({{ '/habilidades/artesania/' | relative_url }}) y material de alquimista para identificar una sustancia alquímica.

## Requisitos

- Competencia **entrenado** en [Artesanía]({{ '/habilidades/artesania/' | relative_url }})
- Material de alquimista

## Tiempo

Identificar un objeto alquímico requiere **10 minutos**.

## Mecánica

Haces una prueba de [Artesanía]({{ '/habilidades/artesania/' | relative_url }}) contra la **CD del nivel del objeto**.

### CD por Nivel

| Nivel del Objeto | CD |
|------------------|-----|
| 0 | 14 |
| 1 | 15 |
| 2 | 16 |
| 3 | 18 |
| 4 | 19 |
| 5 | 20 |
| +1 nivel | +1-2 CD |

## Resultados (Prueba Secreta)

| Resultado | Efecto |
|-----------|--------|
| **Éxito Crítico** | Identificas el objeto y obtienes información adicional (efectos secundarios, usos alternativos) |
| **Éxito** | Identificas el objeto y sus efectos básicos |
| **Fallo** | No logras identificar el objeto |
| **Fallo Crítico** | Identificas erróneamente el objeto |

## Objetos Identificables

- Elixires y pociones alquímicas
- Bombas y venenos
- Herramientas alquímicas
- Sustancias desconocidas
