Utilizas [Arcanos]({{ '/habilidades/arcanos/' | relative_url }}) para preparar un conjuro a partir del libro de conjuros de otro mago.

## Requisitos

- Ser un lanzador de conjuros **arcano preparado** (típicamente mago)
- Competencia **entrenado** en [Arcanos]({{ '/habilidades/arcanos/' | relative_url }})
- Acceso al libro de conjuros de otro mago
- El conjuro debe estar en tu lista de conjuros

## Tiempo

Pedir prestado un conjuro requiere **1 hora** de estudio.

## Mecánica

Haces una prueba de [Arcanos]({{ '/habilidades/arcanos/' | relative_url }}) contra la **CD del nivel del conjuro**.

### CD por Nivel del Conjuro

| Nivel | CD |
|-------|-----|
| 1 | 15 |
| 2 | 18 |
| 3 | 20 |
| 4 | 23 |
| 5 | 26 |
| +1 | +2-3 |

## Resultados

| Resultado | Efecto |
|-----------|--------|
| **Éxito Crítico** | Preparas el conjuro; puedes intentar preparar otro del mismo libro |
| **Éxito** | Preparas el conjuro |
| **Fallo** | No puedes preparar el conjuro hoy |
| **Fallo Crítico** | No puedes preparar ningún conjuro de este libro hoy |

## Notas

- El conjuro **no** se añade permanentemente a tu libro
- Solo puedes usarlo para tus preparativos de **ese día**
- Para añadirlo permanentemente, usa [Aprender un Conjuro]({{ '/reglas/actividades-exploracion/aprender-conjuro/' | relative_url }})
