Utilizas una habilidad adecuada para entender textos arcaicos, esotéricos u obscuros.

## Tiempo

Descifrar una página de texto requiere **1 minuto o más**, dependiendo de la complejidad.

## Habilidades Aplicables

| Tipo de Texto | Habilidad |
|---------------|-----------|
| Textos comunes, idiomas antiguos | [Sociedad]({{ '/habilidades/sociedad/' | relative_url }}) |
| Textos mágicos arcanos | [Arcanos]({{ '/habilidades/arcanos/' | relative_url }}) |
| Textos religiosos o divinos | [Religión]({{ '/habilidades/religion/' | relative_url }}) |
| Textos ocultos o esotéricos | [Ocultismo]({{ '/habilidades/ocultismo/' | relative_url }}) |
| Textos sobre naturaleza o druídicos | [Naturaleza]({{ '/habilidades/naturaleza/' | relative_url }}) |

## Requisitos

- Competencia **entrenado** en la habilidad relevante
- Conocimiento del idioma ayuda pero no es obligatorio

## Resultados (Prueba Secreta)

| Resultado | Efecto |
|-----------|--------|
| **Éxito Crítico** | Comprendes el texto completamente |
| **Éxito** | Comprendes el significado general |
| **Fallo** | No logras descifrar el texto |
| **Fallo Crítico** | Crees haber descifrado el texto pero lo malinterpretas |

## Notas

El DJ hace esta prueba en **secreto**. En un fallo crítico, podrías obtener información incorrecta sin saber que es errónea.
