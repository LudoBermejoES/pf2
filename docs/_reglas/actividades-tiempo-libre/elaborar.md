**Rasgos:** Tiempo Libre, Manipular

Utilizas la habilidad [Artesanía]({{ '/habilidades/artesania/' | relative_url }}) para elaborar objetos a partir de materia prima.

## Requisitos

- Competencia **entrenado** en [Artesanía]({{ '/habilidades/artesania/' | relative_url }})
- **Fórmula** del objeto que quieres elaborar
- **Materias primas** (coste = mitad del Precio del objeto)
- **Herramientas de artesano** apropiadas
- El objeto debe ser de un **nivel igual o menor** que el tuyo

## Proceso

### 1. Preparación

- Obtén la fórmula (compra, aprende o inventa)
- Compra las materias primas (50% del Precio)
- Prepara tu espacio de trabajo

### 2. Trabajo Inicial (4 días)

Inviertes 4 días de trabajo. Al final, haces una prueba de [Artesanía]({{ '/habilidades/artesania/' | relative_url }}) contra la **CD del nivel del objeto**.

### 3. Resultados

| Resultado | Efecto |
|-----------|--------|
| **Éxito Crítico** | Completas el objeto; trabajo adicional reduce coste más rápido |
| **Éxito** | Completas el objeto; puedes trabajar más para reducir coste |
| **Fallo** | No completas el objeto, pero conservas los materiales |
| **Fallo Crítico** | Pierdes el 10% de las materias primas |

### 4. Completar o Continuar

Con éxito, puedes:
- **Pagar el resto** (50% restante del Precio) y terminar inmediatamente
- **Trabajar más días** para reducir el coste restante

## Reducción de Coste

Por cada día adicional de trabajo exitoso, reduces el coste restante según tu nivel y competencia.

| Competencia | Reducción por Día |
|-------------|-------------------|
| Entrenado | Según tabla de Obtener Ingresos |
| Experto | Ganancia de nivel +1 |
| Maestro | Ganancia de nivel +2 |
| Legendario | Ganancia de nivel +3 |
