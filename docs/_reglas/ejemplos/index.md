Los ejemplos de juego te ayudan a entender cómo funcionan las reglas de Pathfinder 2ª Edición en la práctica. Cada ejemplo presenta una situación común con personajes, dados y decisiones reales.

## Cómo Leer los Ejemplos

En cada ejemplo encontrarás:

| Elemento | Descripción |
|----------|-------------|
| **Personajes** | Los PJs y PNJs involucrados con sus estadísticas relevantes |
| **Situación** | El contexto narrativo de la escena |
| **Desarrollo** | La acción paso a paso con tiradas de dados |
| **Reglas Clave** | Las mecánicas importantes que se aplican |

Los iconos de acciones te ayudarán a seguir el flujo:

- {% include accion.html tipo="1" %} = Una acción
- {% include accion.html tipo="2" %} = Dos acciones
- {% include accion.html tipo="3" %} = Tres acciones
- {% include accion.html tipo="reaccion" %} = Reacción

## Consejos para Nuevos Jugadores

### Antes de tu Primera Partida

1. **Lee el ejemplo de combate** — Entender las 3 acciones y la penalización por ataques múltiples es esencial
2. **Familiarízate con los grados de éxito** — Saber cuándo consigues un crítico cambia las decisiones
3. **No memorices todo** — Usa estos ejemplos como referencia durante el juego

### Durante el Juego

- **Pregunta al DJ** — Si algo no está claro, es mejor preguntar que asumir
- **Describe tus acciones** — "Quiero intimidar al guardia" es mejor que "Uso Intimidación"
- **Acepta los fallos** — Los fallos críticos crean momentos memorables
