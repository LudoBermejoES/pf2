## Alzar un Escudo {% include accion.html tipo="1" %}

**Requisitos:** Estas embrazando un escudo.

Colocas el escudo para que te proteja. Cuando has Alzado un escudo, obtienes el bonificador por circunstancia indicado a la CA. Tu escudo permanece alzado hasta el inicio de tu siguiente turno.
