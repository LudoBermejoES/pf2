Este libro es una guía completa para jugar a Pathfinder 2. Está organizado en secciones que cubren todos los aspectos del juego, desde la creación de personajes hasta las reglas avanzadas.

## Estructura del libro

- **Introducción**: Conceptos fundamentales y creación de personajes
- **Tu Personaje**: Atributos, habilidades y mejoras de personaje
- **Ascendencias**: Las diferentes ancestrías jugables
- **Clases**: Las diez clases de personaje disponibles
- **Habilidades**: Acciones y pruebas que pueden realizar los personajes
- **Dotes**: Capacidades especiales y mejoras
- **Equipo**: Armas, armaduras y objetos
- **Conjuros**: Magia y rituales
- **Reglas**: Reglas detalladas para encuentros y combate
- **Apéndices**: Tablas de referencia y glosario

## Cómo navegar

Puedes leer este libro de forma lineal o saltar a las secciones que necesites. Utiliza la barra lateral para navegar rápidamente entre secciones.
