## Habilidades, Atributos Clave y Acciones

| Habilidad | Atributo clave | Acciones no entrenadas | Acciones entrenadas |
|-----------|----------------|------------------------|---------------------|
| **Acrobacias** | Destreza | Mantener el equilibrio ◆, Pasar haciendo acrobacias ◆ | Escurrirse^E^, Maniobrar en vuelo ◆ |
| **Arcanos** | Inteligencia | Recordar conocimiento^G^ ◆ | Aprender un conjuro^E,G^, Descifrar escritura^E,G^, Identificar magia^E,G^, Pedir prestado un conjuro arcano^E^ |
| **Artesanía** | Inteligencia | Recordar conocimiento^G^ ◆ | Elaborar^TL^, Reparar^E^, Identificar alquimia^E^, Obtener ingresos^G,TL^ |
| **Atletismo** | Fuerza | Abrir a la fuerza ◆, Derribar ◆, Empujar ◆, Nadar ◆, Presa ◆, Reposicionar ◆, Salto de altura ◆◆, Salto de longitud ◆◆, Trepar ◆ | Desarmar ◆ |
| **Diplomacia** | Carisma | Causar impresión^E^, Pedir ◆, Reunir información^E^ | - |
| **Engaño** | Carisma | Crear una distracción ◆, Hacerse pasar por otro^E^, Mentir | Fintar ◆ |
| **Interpretación** | Carisma | Interpretar ◆ | Obtener ingresos^G,TL^ |
| **Intimidación** | Carisma | Desmoralizar ◆, Obligar^E^ | - |
| **Latrocinio** | Destreza | Escamotear objeto ◆, Sustraer ◆ | Forzar cerradura ◆◆, Inutilizar mecanismo ◆◆ |
| **Medicina** | Sabiduría | Primeros auxilios ◆◆, Recordar conocimiento^G^ ◆ | Tratar enfermedad^TL^, Tratar heridas^E^, Tratar veneno ◆ |
| **Naturaleza** | Sabiduría | Comandar a un animal ◆, Recordar conocimiento^G^ ◆ | Aprender un conjuro^E,G^, Identificar magia^E,G^ |
| **Ocultismo** | Inteligencia | Recordar conocimiento^G^ ◆ | Aprender un conjuro^E,G^, Descifrar escritura^E,G^, Identificar magia^E,G^ |
| **Religión** | Sabiduría | Recordar conocimiento^G^ ◆ | Aprender un conjuro^E,G^, Descifrar escritura^E,G^, Identificar magia^E,G^ |
| **Saber** | Inteligencia | Recordar conocimiento^G^ ◆ | Obtener ingresos^G,TL^ |
| **Sigilo** | Destreza | Esconder ◆, Movimiento furtivo ◆, Ocultar un objeto ◆ | - |
| **Sociedad** | Inteligencia | Recordar conocimiento^G^ ◆, Subsistir^G,TL^ | Descifrar escritura^E,G^, Falsificar^TL^ |
| **Supervivencia** | Sabiduría | Sentir dirección^E^, Subsistir^G,TL^ | Cubrir rastro^E^, Rastrear^E^ |

## Leyenda

- **TL** - Esta acción de habilidad se utiliza durante el tiempo libre.
- **E** - Esta acción de habilidad se utiliza durante la exploración.
- **G** - Esta es una acción de habilidad general, con una descripción que aparece en la sección de Acciones de habilidad generales, en lugar de en la entrada de la habilidad.

## Símbolos de Acciones

- ◆ = 1 acción
- ◆◆ = 2 acciones
- ◆◆◆ = 3 acciones
