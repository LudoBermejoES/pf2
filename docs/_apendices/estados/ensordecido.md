No puedes oir. Fallas criticamente de forma automatica todas las pruebas de Percepción que requieren que seas capaz de oir. Sufres un penalizador -2 por estatus a las pruebas de Percepción para la iniciativa y para las pruebas que implican el sonido pero que tambien se basan en otros sentidos. Si llevas a cabo una acción con el rasgo auditivo, has de tener éxito en una prueba plana CD 5 o la acción se pierde; haz la prueba después de gastar la acción, pero antes de aplicar cualquier efecto. Eres inmune a los efectos auditivos cuando estas ensordecido.
