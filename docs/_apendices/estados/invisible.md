No puedes ser visto. Estás no detectado para todo el mundo. Las criaturas pueden Buscar para detectarte; si una criatura tiene éxito en su prueba de Percepción contra tu CD de Sigilo, quedas escondido para ella hasta que utilizas Movimiento furtivo para volver al estado de no detectado. Si te haces invisible cuando alguien puede verte, empiezas escondido para dicha criatura (en lugar de no detectado) hasta que utilizas con éxito Movimiento furtivo. No puedes quedar observado mientras eres invisible excepto mediante el uso de aptitudes especiales o magia.
