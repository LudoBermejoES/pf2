Los rasgos son etiquetas que describen propiedades específicas de reglas, criaturas, objetos y aptitudes. Cada rasgo tiene un significado mecánico que afecta cómo funciona algo en el juego.

## Rasgos de Acción

| Rasgo | Descripción |
|-------|-------------|
| [Ataque](/apendices/rasgos/ataque/) | Implica un ataque. Aplica penalizador por ataque múltiple. |
| [Concentrar](/apendices/rasgos/concentrar/) | Requiere concentración y disciplina mental. |
| [Exploración](/apendices/rasgos/exploracion/) | Tarda más de un turno; se usa durante el modo de exploración. |
| [Floritura](/apendices/rasgos/floritura/) | Solo puedes usar una floritura por turno. |
| [Fortuna](/apendices/rasgos/fortuna/) | Tira dos veces y toma el mejor resultado. |
| [Fortuna Adversa](/apendices/rasgos/fortuna-adversa/) | Tira dos veces y toma el peor resultado. |
| [Manipular](/apendices/rasgos/manipular/) | Requiere manipular un objeto o hacer gestos. Provoca ataques de oportunidad. |
| [Movimiento](/apendices/rasgos/movimiento/) | Implica moverse de un espacio a otro. |
| [Reacción](/apendices/rasgos/reaccion/) | Acción que usas fuera de tu turno en respuesta a un desencadenante. |
| [Tiempo libre](/apendices/rasgos/tiempo-libre/) | Dura un día o más; solo se usa durante el tiempo libre. |
