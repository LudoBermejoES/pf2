Una maldición es un efecto que impone alguna aflicción de largo plazo a una criatura. Las maldiciones siempre son mágicas y, típicamente, el resultado de un conjuro o de una trampa. Los efectos con este rasgo sólo se pueden eliminar mediante efectos que designan específicamente como objetivo a las maldiciones.
