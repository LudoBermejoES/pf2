Los elixires son líquidos alquímicos que se activan bebiéndolos. Tienen el rasgo **elixir**.

## Activación de elixires

Activar un elixir requiere una acción de **Interactuar** para beberlo o administrarlo a una criatura adyacente dispuesta o inconsciente. Activa el elixir cuando lo bebes o lo administras.
