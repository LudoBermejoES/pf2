**Nivel** 1; **Precio** 3 mo

<div class="feat-traits-header" markdown="0"><a href="/rasgos/alquimico/" class="feat-trait">Alquímico</a><a href="/rasgos/consumible/" class="feat-trait">Consumible</a></div>

**Uso:** sostenido en 1 mano; **Impedimenta** L

**Activar** {% include accion.html tipo="1" %} Interactuar

---

Este pegamento de secado rápido crea una unión extremadamente fuerte.

**Efecto** Aplicas el pegamento a dos superficies que quieres unir. El pegamento se seca en 1 minuto y crea una unión que requiere un CD 15 de Atletismo para romper (CD 20 para pegamento moderado, CD 25 para mayor).
