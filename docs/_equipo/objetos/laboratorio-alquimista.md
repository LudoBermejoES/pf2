Necesitas un laboratorio de alquimista para Elaborar sustancias alquímicas durante el tiempo libre. Un laboratorio de alquimista mejorado te concede un bonificador +1 por objeto a las pruebas de Artesanía para elaborar sustancias alquímicas.
