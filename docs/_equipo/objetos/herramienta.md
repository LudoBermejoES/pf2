Esta entrada es un cajón de sastre para las herramientas de mano básicas que no tienen un propósito aventurero específico. Una azada, una pala o un martillo pilón son herramientas largas, y un berbiquí, un gancho para hielo o una paleta son herramientas cortas. Una herramienta se suele poder utilizar como arma improvisada, infligiendo 1d4 daño para una corta o 1d6 para una larga.
