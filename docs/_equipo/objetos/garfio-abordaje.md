Puedes lanzar un garfio de abordaje con una cuerda atada al mismo para hacer más fácil una escalada. Para anclar un garfio de abordaje, haz una tirada de ataque a distancia utilizando tu competencia con armas sencillas contra una CD que depende del objetivo, típicamente por lo menos CD 20. Este ataque tiene el rasgo secreto. Con un éxito, tu garfio queda firmemente anclado, aunque con un fallo crítico parece como si fuera a aguantar, pero en realidad se suelta cuando estás a media subida.
