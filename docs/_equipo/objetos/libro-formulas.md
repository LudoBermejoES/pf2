Un libro de fórmulas contiene las fórmulas necesarias para crear objetos diferentes al equipo común de este capítulo; los personajes de la clase alquimista suelen obtener uno gratuitamente. Cada libro de fórmulas puede albergar la fórmula de hasta 100 objetos diferentes.
