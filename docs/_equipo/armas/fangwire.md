Este alambre kobold es fino y difícil de ver, perfecto para una emboscada. El portador hace girar una empuñadura alrededor de un punto vulnerable para que sirva de sujeción del alambre, y luego tira de la otra empuñadura, tensando el alambre y provocando dolorosas laceraciones. La función y el nombre provienen de una trampa basada en alambre llamada "colmillo lento".
