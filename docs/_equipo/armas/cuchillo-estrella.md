A partir de un aro metálico central, se extienden cuatro cuchillas también metálicas y puntiagudas, como las puntas de una rosa de los vientos. Cuando se empuña un cuchillo de estrella por el centro, quien lo hace lo puede utilizar como arma cuerpo a cuerpo. También se puede lanzar a distancias cortas.
