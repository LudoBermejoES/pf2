**Daño** 1d6 Per; **Rango** 50 pies; **Recarga** 1; **Manos** 1; **Impedimenta** L; **Grupo** Hondas; **Categoría** Arma a distancia marcial

---

Los tengu usan estas hondas especializadas para disparar dardos más lejos y con mayor fuerza que cuando se lanzan a mano. Una honda atronadora usa dardos como munición. También puede lanzar dardos de cerbatana como munición pero inflige 1d4 de daño perforante en lugar de 1d6 cuando se usa de este modo.
