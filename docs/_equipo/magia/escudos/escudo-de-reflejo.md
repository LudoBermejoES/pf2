**Dureza** 17; **PG** 68 (UR 34)

**Uso:** sostenido en 1 mano; **Impedimenta** 1

---

La superficie de este escudo es perfectamente pulida como un espejo y brilla con energía arcana.

**Activar** {% include accion.html tipo="reaccion" %} Mando

**Efecto** Cuando eres el objetivo de un conjuro de rango 5 o inferior que requiere una tirada de ataque, puedes usar tu reacción para intentar reflejar el conjuro. Haz una prueba plana CD 10. Si tienes éxito, el conjuro es reflejado hacia el lanzador original.
