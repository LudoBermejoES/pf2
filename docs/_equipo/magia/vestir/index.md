Los objetos de vestir son objetos mágicos que se llevan puestos en el cuerpo. Deben invertirse para funcionar y ocupan una ranura específica del cuerpo.

---

## Ranuras de objetos de vestir

Cada personaje tiene las siguientes ranuras disponibles para objetos de vestir:
- **Cabeza:** coronas, diademas, sombreros, yelmos
- **Cara:** máscaras, gafas, antifaces
- **Cuello:** amuletos, collares, capas
- **Torso:** túnicas, chalecos, vestidos
- **Manos:** guantes, brazaletes, manoplas
- **Dedos:** anillos (2 ranuras)
- **Pies:** botas, sandalias
- **Cintura:** cinturones, fajas
