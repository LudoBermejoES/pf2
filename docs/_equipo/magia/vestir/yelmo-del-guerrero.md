Este yelmo de acero está encantado para mejorar la conciencia en combate.

**Activar** —

**Efecto** Ganas un bonificador por objeto +1 a las pruebas de Percepción para la iniciativa. No puedes quedar desprevenido por flanqueo mientras lleves puesto este yelmo.
