Este anillo de plata tiene grabadas cadenas rotas.

**Activar** {% include accion.html tipo="reaccion" %} Mando

**Efecto** Cuando quedarías agarrado, inmovilizado o paralizado, puedes activar el anillo como reacción. Puedes intentar inmediatamente una prueba para Escapar con un bonificador de circunstancia +2. Puedes usar esta habilidad una vez cada 10 minutos.
