## Notas de uso

1. Copia el texto completo del prompt (todo lo que está entre los backticks)
2. Pega en el chat de Discord de Midjourney o la interfaz web
3. Ajusta el parámetro `--ar 3:4` si deseas otras proporciones
4. Agrega `--quality 2` al final para mayor calidad (usa más créditos)
5. Agrega `--style raw` para resultados menos estilizados
